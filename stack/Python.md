## Python commands

Run program
```python
python program.py
```

Run program in interactive mode
```python
python -i program.py
```

Run in debug mode
```python
python -m pdb program.py
```

## Pockets manage

Upgrade pip
```python
python -m pip install --upgrade pip
```

Show pocket info
```python
pip show pocket_name
```

Download concret pocket version
```python
pip install pocket_name==version
```

## Virtual Environment

Create Virtual Environment
```python
python -m venv venv
```

Activate Virtual Environment
```python
.\venv\Scripts\activate # Windows Powershell
source venv/bin/activate # Linux cmd
```

Deactivate Virtual Environment
```python
deactivate
```
